# Giovanni Maffione — Sito Personale

Sito web personale realizzato in HTML e CSS nell'ambito del Master in Intelligenza Artificiale per il Business.

## 🔗 Link

- **Sito online:** [gmaffione.github.io/giovanni-site](https://gmaffione.github.io/giovanni-site/)
- **Repository:** [github.com/GMaffione/giovanni-site](https://github.com/GMaffione/giovanni-site)

## 📄 Pagine

| Pagina | Descrizione |
|--------|-------------|
| `index.html` | Home — hero, chi sono, competenze, esperienza |
| `cv.html` | Curriculum Vitae completo in HTML |
| `contatti.html` | Form di contatto con validazione |

## 🛠 Tecnologie

- **HTML5** — struttura semantica, Open Graph, favicon SVG
- **CSS3 + Sass** — pattern 7-1, variabili, mixin, nesting BEM
- **Bootstrap 5** — layout responsive, container, normalizzazione
- **GitHub Pages** — deploy e pubblicazione
